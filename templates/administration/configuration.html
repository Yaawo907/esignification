Signature électronique avancée — Yousign
{% if config.yousign_active %}Activé{% else %}Désactivé{% endif %}

Yousign permet aux huissiers de signer les actes numériquement avec une signature électronique avancée (AES) — valeur légale supérieure à une signature manuscrite scannée.

Un seul token API pour toute la plateforme. Les huissiers n'ont pas besoin de compte Yousign.

{% csrf_token %}
⚡ Token API unique géré par l'administrateur. Les huissiers reçoivent un lien de signature par email ; Yousign envoie un code OTP par SMS pour autoriser la signature.
Utilisez Sandbox pour valider avant de passer en prod.
Sandbox : l'email de chaque huissier testeur doit être celui d'un compte de votre organisation Yousign (sinon l'activation de la signature échoue).
Espace Yousign → Paramètres → API. Laisser vide pour conserver.
Ne pas coller l'URL ici. Copiez uniquement la clé secrète HMAC affichée dans Yousign → Developers → Webhooks → votre abonnement → Secret key.
URL webhook à configurer dans Yousign (champ « Endpoint ») : {{ request.scheme }}://{{ request.get_host }}/significations/webhook/yousign/
En local (127.0.0.1), Yousign ne peut pas appeler cette URL. Utilisez ngrok (URL publique HTTPS) ou lancez python manage.py yousign_sync après chaque signature test.
